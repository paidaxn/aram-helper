常见问题

为什么需要管理员权限？

因为英雄联盟客户端以管理员权限运行，只有同权限才能读取它的进程信息。这是与 LCU 交互的唯一前提。

会被封号吗？

不会。LCU 是英雄联盟客户端本身暴露的本地接口，Rank-Analysis、WeGame 等都在用。我们只读取数据、不修改客户端、不干预游戏行为。

支持哪些模式？

优先适配海克斯大乱斗（ARAM）。匹配、排位、无限火力、斗魂竞技场都可用，只要 LOL 能记录伤害数据。

历史对局的分队为什么可能不准？

"楼层"信息只在选英雄阶段可以捕获。安装本工具之前打过的历史对局没有这份数据，只能用默认顺序推测。装了之后打的每一局，分队都是准确的。

能设置红包金额吗？

下个版本加。目前只判定"谁发"，金额你们自己商量。

Mac 或手机能用吗？

目前只有 Windows 版。英雄联盟客户端主要在 Windows 上，暂时没有其他平台计划。
