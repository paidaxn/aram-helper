1

下载并安装

2

以管理员权限启动

右键图标 → "以管理员身份运行"。
需要管理员权限读取 LOL 进程信息。

3

打开英雄联盟

打开 LOL 客户端，窗口会自动出现在屏幕右下角并显示"已连接"。接下来正常打游戏，每局结束自动弹出红包局结算。

常见问题

为什么需要管理员权限？

因为英雄联盟客户端是以管理员权限运行的，只有我们的工具也以管理员权限启动，才能读取它的进程信息（端口 + token）。这是唯一的使用限制。

会不会被封号？

不会。LCU API 是英雄联盟客户端暴露的本地接口，Rank-Analysis、WeGame 等第三方工具都在使用，腾讯默许社区使用。我们只读数据不修改，不涉及任何外挂行为。

支持哪些游戏模式？

优先适配海克斯大乱斗（ARAM），其他模式（匹配、排位、无限火力、斗魂竞技场等）也可用，只要能获取伤害数据即可。

好友检测怎么工作的？

通过 LCU API 读取你的英雄联盟好友列表，在对局队友中匹配，自动识别一起玩的好友。无需手动配置。

历史对局的分队为什么不准？

"楼层"信息只在选英雄阶段可以获取，app 安装前打过的历史对局没有记录。安装后打的对局，楼层都是准确的。

可以设置红包金额吗？

下个版本会加。目前只显示谁发红包，金额你们自己商量。
